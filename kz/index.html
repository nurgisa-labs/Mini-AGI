күн 7контур көрсетеміз

Жауаптар, тапсырмалар және хаттамалар нобай ретінде дайын. Сыртқы әрекет — тек approval-дан кейін.

./loop

Бес жоғалған орынның орнына бір ағын.

capture

хабарлама, кездесу, файл немесе сілтеме жұмыс контурына түседі.

context

Mini-AGI жобаны, адамдарды, өткен шешімдерді және шектеулерді табады.

draft

хаттама, тапсырма, жауап немесе қысқа жазба дайындалады.

memory

расталған шешім жоба жадына қайта жазылады.

./ережелер

Автономия — тек тексеруден кейін.

дереккөз

әр фактінің сілтемесі, файлы немесе “тексерілмеді” белгісі бар.

тәуекел

ақша, клиенттер, жарияланымдар және заң сұрақтары автоматты түрде кетпейді.

approval

сыртқы әрекет пен соңғы мәтінді адам бекітеді.
